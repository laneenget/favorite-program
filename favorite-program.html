C# Matching Game

This C# program is a timed matching game with varying difficulty levels. It takes your time, multiplies it by a static number to add a score to the database. The tricky part to this program is that for each round, the pairs are randomized. The card images are stored in a list and some extensive logic is involved to not only pull a pair from the list, but to make sure that same pair is not pulled a second time.
